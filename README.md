# quantum-tunneling
this is a quantum-tunneling simulation, that i made because i was bored :)
its not ideal for 100%, but its not made without any physics knowledge
so, if you want to learn programming or physics you can play it or modify it for free
it have MIT license, so go ahead

## compiling

if you are on windows, run the build.bat
if you are on linux, run the build.sh

the only dependency is raylib.
